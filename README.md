# Movie Ticket Booking System

**Author:** Nitheesh S  
**Language:** Java (console application)  
**Short:** Simple OO Java program to manage movie listings, bookings and snacks billing.

## Project Overview
A console-based movie ticket booking system. Features:
- Movie listing with available seats
- Book tickets and update seat count
- Order snacks and include in total bill
- View bookings and upcoming movies
